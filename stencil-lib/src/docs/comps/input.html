Basic Inputs

Canopy inputs display a form input for a basic text/number/email/password field. They are referenced using the custom HTML element.

Canopy forms use a custom validation factory to provide built-in validator functions for your inputs. They can also be extended upon to create your own validators.

Learn more about form validation in Canopy.
