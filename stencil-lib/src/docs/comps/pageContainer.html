Page Containers

Canopy page container components are stylised layout components for blog/doc style content. They are used to provide a basic set of padding/margins based on screen size. They are referenced using the custom HTML element. This component utilises the slot element, allowing you to render your own HTML in the content.

Custom Styles

Through the use of CSS variables, you can customise Canopy components to achieve the look and feel you want. There are currently no styling variables for this component. You can learn more about Canopy colours and root configuration here.
