Textarea Inputs

Canopy textarea inputs display a form input for large amounts of text, similar to the native element. They are referenced using the custom HTML element.

Textarea inputs will autosize to fit the text content and make sure it is all visible. You can use the and properties to set a limit on how short or tall a textarea can become.
